Wikiful is a plugin that integrates the powerfull Mediawiki software with Wordpress.

== Description ==

Wikiful is a plugin that seamlessly integrates Mediawiki, the world's most popular wiki platform with WordPress. 

In this first release, the plugin allows you to view a Mediawiki in a Wordpress page. In the future we will also add single-sign-on features as well as the possibility to edit your wiki from the confort of your Wordpress installation.
 
== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or install via the Wordpress plugins control panel
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Wikiful settings menu and configure the plugin options.
4. Install the plugin by clicking the Install button. This will create a new page titled Wikiful through which you can access your wiki.
5. Activate the sidebar widgets if you want the sidebar navigation and search functions

Please visit the [Support Forums](http://wordpress.org/tags/wikiful?forum_id=10 "Support Forums") for more information and support.
